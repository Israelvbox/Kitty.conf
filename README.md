## Instalacion
1. Copy and paste.

```shell
sudo apt install kitty zsh curl git -y
git clone https://github.com/Israelvbox/Kitty.conf && cd kitty.conf
sudo chmod +x zap.sh && ./zap.zsh
sudo cp kitty.conf ~/.config/kitty/kitty.conf
sudo rm -r ~/Kitty.conf
```
